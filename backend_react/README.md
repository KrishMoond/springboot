# React Backend API

Spring Boot REST API backend for React frontend application.

## Features

- RESTful API endpoints
- Spring Boot Web
- CORS configuration for React
- JSON response handling

## Tech Stack

- **Spring Boot**: 3.5.11
- **Spring Web**: REST API
- **Java**: 17

## Build & Run

```bash
# Build
mvn clean install

# Run
mvn spring-boot:run
```

## Default Port

Application runs on: `http://localhost:8080`

## API Endpoints

Configure your REST controllers to serve React frontend.
